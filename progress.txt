## Codebase Patterns
- Preload script at `src/main/preload.ts` exposes `window.prominenceAPI` for renderer IPC ‚Äî always use `contextBridge.exposeInMainWorld`
- IPC handlers registered in `src/main/index.ts` inside `app.whenReady().then()` block
- Main process TypeScript files go in `src/main/`, renderer HTML/CSS in `src/renderer/`
- Use `nativeImage.createFromBuffer()` for programmatic tray icons (no external PNG needed)
- Electron `window-all-closed` handler must not call `app.quit()` for tray apps
- TypeScript compiles from `src/` to `dist/`, entry point is `dist/main/index.js`
- For transparent frameless windows: `html,body { background: transparent }`, visible background on a container div
- Overlay window is hidden/shown (not destroyed/recreated) for performance ‚Äî use `getOverlayWindow()` to access it
